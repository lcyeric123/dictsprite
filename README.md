# DictSprite 极简离线中英繁词典系统
完全开源 | 零广告 | 无联网请求 | 无数据上传 | 纯内网运行 | 无任何多余组件

## 项目介绍
市面所有词典软件全部存在广告、弹窗、资讯推送、后台隐私收集、强制联网、后台上传查询记录、捆绑多余功能等问题。
DictSprite 只做唯一一件事情：字词查询，除此之外全部功能全部砍掉。
整个项目只有两个文件：index.html + cedict.txt，不需要编译、不需要运行库、不需要Node、不需要后端、不需要数据库。
全程不会向外发送任何一条网络请求，所有数据全部在本机内存完成运算，隐私性极高，可在内网局域网永久部署使用。

自动识别三种语言输入：简体中文、繁体中文、英文模糊检索，输入文字实时返回结果，不需要按下回车键。

## 核心特性
- 全程零广告、无开屏、无弹窗、无资讯流、无推荐内容
- 全程离线运行，不会访问外网、不会上传任何查询记录
- 无Cookie写入、无本地日志、无数据采集、无隐私抓取
- 深色精致UI，界面极简，没有多余按钮、菜单、跳转页面
- 自动语种判断，无需手动切换中英繁模式
- 英文支持模糊包含检索，输入片段英文即可匹配整条释义
- 可以部署在内网服务器，局域网所有电脑浏览器打开即可使用
- 完全开源，可自由修改界面、逻辑、样式，允许私人二次定制，禁止打包商业化售卖

## 文件构成
项目总共仅有两个文件，结构极其简单：
1. index.html  主程序界面与查询逻辑
2. cedict.txt  开源CC-CEDICT中英词库，放置在html同级文件夹

## 部署方式（两种方案）
### 方案一：局域网内网部署（推荐正式使用）
1. 将index.html和cedict.txt放在同一个文件夹内
2. 在文件夹内启动静态网页服务，debian一键命令：
```bash
python3 -m http.server 8080 --bind 0.0.0.0
```
 
3. 局域网内所有设备打开：http://?܍务器IP:8080 即可使用
4. 后续更新词库只需要替换cedict.txt，无需改动网页文件
 
方案二：本地单机使用
 
Windows无法直接双击打开，浏览器会触发本地跨域安全限制，因此单机建议使用微型静态服务打开。
 
内置逻辑说明
 
1. 程序启动一次性读取cedict.txt载入内存，之后不再读取硬盘文件
2. 建立简体哈希表、繁体哈希表、英文释义数组三个索引
3. 通过内置繁简字规则自动判断输入文字语种，不需要OpenCC等第三方转换库
4. 简体输入直接哈希秒查，繁体输入哈希秒查，英文全文模糊匹配释义字段
5. 所有运算全部在浏览器前端完成，不需要后端参与计算，速度极快
