# 詞典精靈 DictSprite
本軟件採用 NodeGUI 原生 Qt 控件開發，全程沒有 Electron、Chromium、網頁引擎，屬於純原生Windows程序。

## 功能
1. 自動識別輸入文字：簡體中文、繁體中文、英文三種格式
2. 輸入任意一種文字，自動返回另外兩種文字、拼音、英文釋義
3. 分為圖形界面版本、CLI後台引擎版本，可以嵌入 PASS System 內部調用
4. 所有查詢全部在內存完成，不會聯網、不會上傳任何文字、不會寫入日誌
5. 安裝包體積穩定控制在9MB以內，綠色攜帶，可隨意搬移
